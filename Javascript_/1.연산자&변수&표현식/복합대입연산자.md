```
var value = 10;
value += 10;
alert(value); //20
```

```
value += 10;
같은 의미
value = value + 10;
```

| 연산자 |          설명         |        사용 예        |
|:------:|:---------------------:|:---------------------:|
| !      | 논리 부정 연산자(not) | !(3>2) : false        |
| &&     | 논리곱 연산자(and)    | 3>2 && 1>0 : true     |
|   ||   | 논리합 연산자(or)     | 1>10 \|\| true : true |
