### 이스케이프 시퀀스
- 문자나 키보드에서 직접 입력할 수 없는 문자를 표현하기 위한 방법
- 줄 바꿈 문자, 탭 문자 등은 문자열에 그대로 추가할 수 없다.
- \ (역슬래시) 를 이용해서 특정 제어 코드나 문자를 표현 할 수 있다.
예시
```
"i'm going to learn javascript."
"i\'m going to learn javascript."
```
    - 종류
    - \b : 백스페이스
    - \t : tab
    - \n : 개행
    - \f : 페이지 열기
    - \r : 복귀
    - \' : 작은 따옴표, \" : 큰 따옴표
    - \\ : \문자
    - \0 : \ NULL문자
